Note about driver verisons: There are two sets of Windows drivers, one that is based on the Cypress USB driver stack and the other is based on the libusb/WinUSB driver stack. To make the USB backend in the SDK uniform across all platforms we are converging to using only libusb. However libusb on Windows relies on WinUSB kernel stack provided by Microsoft, which was not mature in Windows 7. For this reason, Windows 7 users should install the 1.0.16 driver which uses the old Cypress USB backend. Windows 8 users should have no problem with libusb, but either version will work.
