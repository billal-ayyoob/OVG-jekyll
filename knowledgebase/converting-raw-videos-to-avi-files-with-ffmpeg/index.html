Converting RAW videos to AVI files with ffmpeg

Last Updated: 6 years ago in Indigo Tools

The recording functions in Indigo Tools record to a proprietary RAW video format. As of version of 1.2 of Indigo Tools, there is no way to directly emit AVI files from the software. We are working on this feature as well as including support for reading the RAW video files using the Indigo SDK.

The software does provide a function to export RAW videos to image sequences, and using this you can generate AVI files easily. There is just an extra step of exporting PNG or JPG files for each frame first.

STEP 1: open the recording in Indigo Tools, by selecting the recording list button in the main screen. When the popup appears, click on the video you want to convert.

STEP 2: The video file will open. Now click the export button in the toolbar to begin the process of exporting to image files.

STEP 3: When the export options dialog box appears, click Browse to select the output folder and then click OK.

STEP 4: The files will now convert and appear as a set of files in a subdirectory of the directory you selected. The name of first subdirectory will be the the video’s name followed by its unique ID, and then another subdirectory called “images”.

 

 

STEP 5: The next step will be to convert the still images to an AVI file using the ffmpeg command line tool. If you already have ffmpeg installed you can skip this step.

You can download Windows binaries of ffmpeg here. We’ll assume you unpacked these files to c:\ffmpeg. You’ll want to open a terminal by clicking Start, Run, and running the “cmd” program.

 

Then add the ffmpeg directory to your path by entering

@set PATH=c:\ffmpeg\bin;%PATH%

at the prompt.

 

STEP 6: go to the directory where the videos got unpacked by entering the directory just underneath where the images were put.

At the prompt, enter:

e:
cd E:\exports\hackaday_2_take4-3eaf1bc7bc6c99a3ae4a13f08d7ff884

STEP 7: now you can run ffmpeg to convert the image files into a single AVI. ffmpeg is a huge program with support for a lot of codecs and video/audio containers. You’ll want to refer to the full ffmpeg documentation on the ffmpeg web site here.

Here we’ll show just the basic form of the command to create an AVI file. At the prompt, enter:

ffmpeg -f image2 -r 60 -i images/%d.jpg -b:v 500k test60.avi

You’ll want to play around with the bitrate option “-b:v” and other codec options to control the output format and quality.

Once ffmpeg will process the files you will be left with a single avi file that has a frame per each image in the images subdirectory.

 

You should be able to open the video with Windows Media Player or VLC, which you can download here.
