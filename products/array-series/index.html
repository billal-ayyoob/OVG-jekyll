 Array Series – Machine vision board cameras

The Array is a series of board cameras that capture high resolution images at a high frame-rate.  Designed to be used as a single unit or in an array as a complete multi-camera capture solution with each camera synchronized across a USB3 bus, as well as external sync via input and output trigger strobes, and included multi-camera geometric calibration. Each sensor can capture high resolution images at high frame rates (60FPS+ and higher), with a USB3 interface. Windows 7/8 software tools and open source SDK, Linux SDK, and ROS module provided.

Available models: bu3ar0330, bu3mt9m031m, bu3mt9m031c, bu3mt9j003, bu3imx174m, bu3imx174c, bu3cmv4000m, bu3cmv4000c.

  • See our technical specs for resolution + framerate ranges
  • Low latency
  • GPIO: 2x opto-isolated inputs, 2x opto-isolated outputs. Camera trigger input and output strobe.
  • On-board color processing: debayer, gamma, brightness, digital gain, LUT remap
  • 12-bit raw mode, 10-bit raw mode, 8-bit raw mode
  • Synchronize multiple cameras over USB3 bus
  • Multi-camera calibration
  • Multi-camera recording
  • C, C++, GenICam/GenTL, ROS
  • Windows 7, 8, 10, Linux 3+

Array Series – Machine vision board camera

Array is a series of USB 3.0 camera for machine vision that can be used standalone or in an array as a complete multi-camera capture solution. Each Array sensor is a board camera that can capture high resolution images at high frame rates, returning raw image data to the host system by USB3. The cameras were designed to be deployed in multiples, and supports easy synchronization over the USB3 bus with other Occam cameras as well as external sync via input and output trigger strobes.

Windows software tools provide full multi-camera geometric calibration and recording support. An open source Windows/Linux SDK and ROS module are provided, as well as GenICam driver for wide base of third party software compatibility.

Designed for use in machine vision, manufacturing and industry, as well as research and development and computer vision / machine vision end uses.
